title: "Getting started"
description: "Checklist rapido para entrar em operacao no Ardra."
---

## 1. Solicite acesso

- Entre em contato via `hello@ardra.xyz` ou use o formulario interno.
- Recebera convite para o ArdraHubbot e para o dashboard web.

## 2. Configure contas

1. Crie seu usuario (SIWS ou email + senha).
2. Informe refCode (se voce veio por um squad) para que pontos sejam atribuidos.
3. Ative 2FA.

## 3. Vincule wallets e APIs

| Tipo | Onde informar | Observacoes |
| --- | --- | --- |
| Solana | Dashboard > Wallets | A chave privada nao sai do backend. |
| Hyperliquid/Backpack APIs | Dashboard > Integrations | Precisamos das scopes de trade/leitura. |

## 4. Escolha um preset

<Card title="Recomendado" icon="thumbs-up">
  Use o preset `Starter` do venue que voce quer explorar. Ele limita tamanho e parametros conservadores.
</Card>
<Card title="Personalizado" icon="sliders">
  Squads com contrato podem solicitar presets customizados revisados pela equipe do Ardra.
</Card>

## 5. Executar e observar

- Inicie o bot pelo dashboard ou comando `/start` no ArdraHubbot.
- Acompanhe funding, fills e alertas na aba Telemetria.
- Se ocorrer erro, exporte os logs e abra um ticket via Discord ou e-mail.

## 6. Proximo passo: rede

Depois da primeira semana, gere seu link de referrals e monte uma base de pilotos. Use o dashboard (`/dashboard`) para ver o que cada piloto entrega e ofereca presets alinhados.
