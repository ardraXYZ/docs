## Componentes do score

- **Pontos diretos:** cada venue tem uma tabela de conversao baseada em volume, numero de ordens e tempo conectado.
- **Referral points:** 10% do que um referral direto gera entra no seu total. Se voce tiver squads abaixo de voce, esse numero escala rapido.
- **Fees creditados:** 20% do fee liquido do referral e creditado em USD equivalente para acompanhar ROI.

## Como cadastrar referrals

1. Gere seu link/QR code dentro do Ardra (Dashboard > Referrals).
2. Peça para o novo piloto criar conta passando pelo link ou informando seu refCode.
3. Assim que o piloto terminar o onboarding, ele aparece no Dashboard com status `pending` ate realizar a primeira execucao.

## KPI sugeridos

| Indicador | Como usar |
| --- | --- |
| Pts diretos x rede | Mantem proporcao saudavel (ideal >= 60% direto). |
| Fee share por referral | Identifica quem precisa de apoio para configurar presets melhores. |
| Tempo medio para ativar | Mede quantos dias entre convide e primeira execucao. |
